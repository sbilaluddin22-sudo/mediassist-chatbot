# MediAssist AI — React Application

A medical information chatbot powered by the Anthropic Claude API.

## Getting Started

### Prerequisites
- Node.js 16+ installed
- An Anthropic API key (get one at https://console.anthropic.com)

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start
```

The app will open at http://localhost:3000

### Build for production

```bash
npm run build
```

## Usage

1. Paste your Anthropic API key in the sidebar
2. Click **Connect API Key**
3. Ask any medical question in the chat

## Pages

- **Chat** — The main medical AI chat interface
- **How It Works** — Step-by-step guide, capabilities, FAQ, and disclaimer

## ⚠️ Disclaimer

This application provides general health information only. It is not a substitute for professional medical advice, diagnosis, or treatment. Always consult a qualified healthcare provider.
